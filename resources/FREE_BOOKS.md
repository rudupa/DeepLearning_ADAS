# 📚 Free Books & Textbooks for Deep Learning & ADAS

A curated list of **free, legally available** books for learning deep learning and its application to autonomous driving.

---

## 🧠 Deep Learning Fundamentals

### 1. Deep Learning — Ian Goodfellow, Yoshua Bengio, Aaron Courville
- **URL:** https://www.deeplearningbook.org/
- **Level:** Intermediate
- **Why read it:** THE definitive deep learning textbook. Covers math foundations, MLPs, CNNs, RNNs, optimization, regularization.
- **Best chapters:** Ch. 6 (Deep Feedforward Networks), Ch. 9 (CNNs), Ch. 10 (RNNs)

### 2. Neural Networks and Deep Learning — Michael Nielsen
- **URL:** http://neuralnetworksanddeeplearning.com/
- **Level:** Beginner
- **Why read it:** Excellent intuitive explanations with interactive visualizations. Perfect first book.

### 3. Dive into Deep Learning (D2L)
- **URL:** https://d2l.ai/
- **Level:** Beginner to Intermediate
- **Why read it:** Interactive book with runnable code (PyTorch, TensorFlow, JAX). Covers CNNs, RNNs, Transformers, GANs. Actively maintained.
- **Best for:** Hands-on learners who want to code while reading.

### 4. Understanding Deep Learning — Simon J.D. Prince (2023)
- **URL:** https://udlbook.github.io/udlbook/
- **Level:** Intermediate
- **Why read it:** Modern textbook with excellent visualizations. Covers transformers, diffusion models, and latest architectures.

### 5. The Little Book of Deep Learning — François Fleuret (2023)
- **URL:** https://fleuret.org/francois/lbdl.html
- **Level:** Beginner to Intermediate
- **Why read it:** Concise (200 pages), beautifully illustrated. Perfect quick reference.

## 🔢 Mathematics for Deep Learning

### 6. Mathematics for Machine Learning — Deisenroth, Faisal, Ong
- **URL:** https://mml-book.github.io/
- **Level:** Beginner to Intermediate
- **Why read it:** Linear algebra, calculus, probability — all the math you need for DL.

### 7. The Matrix Cookbook — Petersen & Pedersen
- **URL:** https://www.math.uwaterloo.ca/~hwolkowi/matrixcookbook.pdf
- **Level:** Reference
- **Why read it:** Quick reference for matrix derivatives, identities, and operations.

---

## 🤖 Machine Learning (Broader Context)

### 8. Pattern Recognition and Machine Learning — Christopher Bishop
- **URL:** https://www.microsoft.com/en-us/research/publication/pattern-recognition-machine-learning/
- **Level:** Advanced
- **Why read it:** Comprehensive ML textbook with Bayesian perspective. Deep theoretical foundation.

### 9. Probabilistic Machine Learning — Kevin Murphy (2022, 2023)
- **URL:** https://probml.github.io/pml-book/
- **Level:** Intermediate to Advanced
- **Why read it:** Modern, comprehensive. Two volumes covering foundations and advanced topics.

### 10. An Introduction to Statistical Learning (ISLR)
- **URL:** https://www.statlearning.com/
- **Level:** Beginner
- **Why read it:** Gentle introduction to ML concepts. Great for building intuition.

---

## 🚗 Computer Vision & ADAS-Specific

### 11. Computer Vision: Algorithms and Applications — Richard Szeliski (2022)
- **URL:** https://szeliski.org/Book/
- **Level:** Intermediate to Advanced
- **Why read it:** Comprehensive CV textbook. Covers 3D vision, motion estimation, recognition — all relevant to ADAS.

### 12. Multiple View Geometry in Computer Vision — Hartley & Zisserman
- **URL:** Available on author's website
- **Level:** Advanced
- **Why read it:** Essential for understanding 3D reconstruction, camera geometry — critical for autonomous driving.

---

## 🤗 NLP & Language Models

### 13. Speech and Language Processing — Dan Jurafsky & James Martin
- **URL:** https://web.stanford.edu/~jurafsky/slp3/
- **Level:** Intermediate
- **Why read it:** Covers NLP foundations, word embeddings, transformers, LLMs. Great for understanding the language side of VLMs.

### 14. Natural Language Processing with Transformers — HuggingFace
- **URL:** https://transformersbook.com/ (partial free content)
- **Level:** Beginner to Intermediate
- **Why read it:** Practical guide using HuggingFace ecosystem.

---

## 🔧 Practical & Framework-Specific

### 15. PyTorch Tutorials (Official)
- **URL:** https://pytorch.org/tutorials/
- **Level:** All levels
- **Why read it:** Official tutorials, always up-to-date. Essential reference.

### 16. Hugging Face Course
- **URL:** https://huggingface.co/learn
- **Level:** Beginner to Intermediate
- **Why read it:** Free course on transformers, NLP, and multimodal models.

### 17. Fast.ai — Practical Deep Learning for Coders
- **URL:** https://course.fast.ai/ (+ free book)
- **Level:** Beginner
- **Why read it:** Top-down approach — build things first, understand theory later.

## 📖 Suggested Reading Order

| Stage | Book | Focus |
|-------|------|-------|
| 1️⃣ Start here | Neural Networks & DL (Nielsen) | Build intuition |
| 2️⃣ Get hands-on | Dive into Deep Learning (D2L) | Code while learning |
| 3️⃣ Go deeper | Deep Learning (Goodfellow) | Theoretical foundations |
| 4️⃣ Modernize | Understanding Deep Learning (Prince) | Latest architectures |
| 5️⃣ Specialize | Computer Vision (Szeliski) | Vision for ADAS |
| 6️⃣ Multimodal | HuggingFace Course | VLMs and LLMs |
